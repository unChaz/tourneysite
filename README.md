## Getting Started

### Requirements
- Ruby ~2.3
- Rails ~4.2
- PostgreSQL

## Commands
- Install Dependencies: `bundle install` and `bower install`.
- Database Setup: `rake db:create` and `rake db:migrate`.
- Run the Server: `rails s`
